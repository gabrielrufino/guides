# Guides by Gabriel Rufino

Technical guides on software architecture, databases, and application development.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 24 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

Clone the repository and install the dependencies:

```bash
git clone https://github.com/gabrielrufino/guides.git
cd guides
npm install
```

## 🛠️ Basic Scripts

| Script | Description |
|--------|-------------|
| `npm run docs:dev` | Starts the development server with hot-reload. |
| `npm run docs:build` | Generates the static site in the `src/.vitepress/dist` folder. |
| `npm run docs:preview` | Previews the build output locally. |

## 📁 Repository Structure

- `src/`: Contains all technical documentation and guides.
- `src/.vitepress/`: VitePress configuration, theme, and assets.
- `src/*/`: Technical guides on different topics.
